Apa itu CSS

CSS atau Cascading Style Sheet adalah dokumen web yang memiliki fungsi mengatur tampilan dokumen web lain yaitu HTML. Beberapa contoh pemanfaatan CSS pada HTML adalah untuk mengatur warna latar belakang, gaya huruf, jarak antar elemen, dan masih banyak lagi.
CSS Terdiri atas kumpulan kode-kode yang keberadaanya tidak menggantikan html melainkan sebagai pelengkap agar tampilan dokumen HTML menjadi lebih indah dan dinamis. Berikut ini adalah contoh dokumen css

Bagaimana menggunakan CSS

ada 3 cara menggunakan CSS :

  1. inline Styles : Menuliskan code CSS langsung pada tag yang akan diubah atau diatur stylenya
  2. Internal/Embedded : Menuliskan kode CSS pada bagian header code HTML
  3. Eksternal/Linked Style : Menulisian kode css pada file terpisah dan dipanggil melalui tag link dari documen HTML
